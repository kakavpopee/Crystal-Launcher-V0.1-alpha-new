## 🚀 About

Crystal Launcher is an open-source Minecraft Java launcher for Android focused on providing a clean UI, excellent performance, and modern features while remaining compatible with the Minecraft Java ecosystem.

## ✨ Features

- 🎮 Minecraft Java Edition on Android
- ☕ OpenJDK Runtime
- 🌐 Microsoft Authentication
- 👤 Ely.by Authentication
- 🖥️ Local Account Login
- 🧩 Forge Support
- 🧵 Fabric Support
- ⚙️ NeoForge Support
- 📦 Modpack Support
- 💎 Crystal Launcher Interface
- 🚀 Performance Optimizations
